Para ejecutar el programa:
	1. Llegue a donde se encuentra este documento por medio de directorios
	2. Luego correr el siguiente comando en la terminal sin las comillas "octave --no-gui Diferencia_Numeros.m"
    3. El programa le va a pedir que ingrese los datos y como quiere escoger el programa, 1 para correrlo de 
       forma predeterminada y 2 para escoger los parametros para ingresarle a la funcion.
    4. A la hora que aparezca la grafica esta no se cerrara hasta dar un enter e la terminal.
